This project focuses on **detecting football players in match clips** using State-Of-The-Art YOLOV5 object detection model which is   further trained to improve performance on football dataset. Players are also assigned to their respective teams using the color of their tshirt and Kmeans clustering. 

## Key Features
- Automatic player detection in football videos  
- Frame-by-frame inference using a deep learning model  
- Annotated video output with detected players
- Player team assignment using Kmeans clustering
